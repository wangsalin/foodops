# Roadmap

This roadmap is scoped to the community core. Enterprise integrations should remain outside this repository unless they are designed as optional plugin boundaries.

## V0.1 Stabilize The Local Loop

- Improve demo seed coverage across stores, products, alerts, and task feedback.
- Add focused backend tests for import normalization and alert-to-task behavior.
- Add frontend smoke tests for dashboard, alerts, tasks, and H5 feedback.
- Document import file templates.

## V0.2 Contributor Experience

- Add local development scripts for one-command setup.
- Provide sample CSV/XLSX files for each import type.
- Add screenshots generated from the demo seed.
- Add a lightweight plugin proposal template.

## V0.3 Operations Depth

- Add more configurable local alert rules.
- Improve task review history and audit filters.
- Add exportable operations reports.
- Harden permissions around master data and audit logs.
