## Thread

1/ We just open-sourced FoodOps Community.

It is a self-hosted operations loop for chain food & beverage teams, focused on the daily workflow after local business data is exported.

https://github.com/wangsalin/foodops

2/ The community core is intentionally small:

- master data
- manual imports
- dashboard
- rule-based alerts
- task dispatch
- H5 feedback
- notifications
- audit logs

3/ It is not a delivery platform, ERP replacement, or enterprise connector bundle.

The first goal is simpler:

make the local loop easy to run, inspect, test, and improve.

4/ Stack:

- FastAPI
- SQLAlchemy / Alembic
- PostgreSQL
- Redis
- Next.js
- Ant Design

5/ Good first issues are ready:

- import templates
- backend tests for normalizers
- alert-to-task tests
- frontend smoke tests
- richer demo seed

Contributors and operators are welcome.

https://github.com/wangsalin/foodops/issues
